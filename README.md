# Password Security Analyzer

A Flask-based cybersecurity web application that analyzes password strength and checks exposure against a known breach dataset.

##  Live Demo
https://password-security-analyzer.onrender.com

##  Features
- Password strength analysis
- Visual security dashboard
- Breach exposure check (local dataset)
- Improvement suggestions
- Clean UI with HTML/CSS

## Tech Stack
- Python
- Flask
- HTML
- CSS
- GitHub
- Render

##  Disclaimer
This project uses a sample breach dataset for educational purposes. No passwords are stored or logged.
